Flask Leaflet ChartJS

Vizualization Demo

ChartJS is a JavaScript library consisted of a collection of chart types that can be integrated into a data vizualization web application. Leaflet is a JavaScript Library for handling maps. This demo utilizes BootStrap, jQuery, ChartJS, and Leaflet for front-end rendering and Python Flask and MySQL for handling back-end JSON and GeoJSON object generation and data storage.
